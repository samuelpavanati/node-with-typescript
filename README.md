# Commands
```
pnpm init
pnpm i @types/node typescript tsup tsx -D
npx tsc --init
pnpm run dev
npx tsc (warning - converte de ts para js)
pnpm run build
npx tsx src/index.ts (executa o projeto direto na linha de comando)
tsc (converte em .js na pasta configurada no outDir)
```
